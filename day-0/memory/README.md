## Быстрая шпаргалка

```
ПАМЯТЬ (memory) = СТЕК (stack) + КУЧА (heap)

СТЕК (stack):
• Быстрый
• Маленький
• Автоматический
• Для чисел, да/нет

КУЧА (heap):
• Медленнее
• Большой
• Нужен контроль
• Для текста, списков

RUST ЗАЩИЩАЕТ:
• Один хозяин (owner) у данных
• Хозяин "ушёл" = данные удалились
• Нет утечек памяти (memory leak)
```
