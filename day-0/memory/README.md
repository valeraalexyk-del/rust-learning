## Быстрая шпаргалка

```
ПАМЯТЬ = СТЕК + КУЧА

СТЕК:
• Быстрый
• Маленький
• Автоматический
• Для чисел, да/нет

КУЧА:
• Медленнее
• Большой
• Нужен контроль
• Для текста, списков

RUST ЗАЩИЩАЕТ:
• Один хозяин у данных
• Хозяин "ушёл" = данные удалились
• Нет утечек памяти
```
