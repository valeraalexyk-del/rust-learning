## Что такое стек?

**Стек** (англ. **stack** — стопка) — как стопка тарелок.

> В программировании стек работает по принципу **LIFO** — Last In, First Out (последний вошёл — первый вышел).

```
    ┌─────────┐
    │ тарелка │ ← кладём сверху
    ├─────────┤
    │ тарелка │
    ├─────────┤
    │ тарелка │ ← снизу первая
    └─────────┘

Кладём СВЕРХУ, берём тоже СВЕРХУ.
```

---

## Как программа использует стек?

При вызове функции создаётся **стековый кадр** (англ. **stack frame**) — область памяти для переменных этой функции.

```rust
fn main() {
    let a = 5;    // положили в стек (stack allocation)
    let b = 10;   // положили в стек
}
```

```
Стек:
┌──────────┐
│  b = 10  │ ← сверху
├──────────┤
│  a = 5   │ ← снизу
└──────────┘
```

Когда программа заканчивается — стек очищается сам.

## Плюсы

| Плюс | Почему |
|------|--------|
| Очень быстрый | Просто кладём/берём сверху |
| Автоматический | Сам очищается |
| Простой | Всё предсказуемо |

---

## Минусы

| Минус | Почему |
|-------|--------|
| Маленький | Всего 1-8 МБ |
| Фиксированный размер | Нужно знать размер заранее |

---

## Что хранить в стеке?

- Числа (`i32`, `u8`, `f64`)
- Да/нет (`bool`)
- Символы (`char`)
- Маленькие данные известного размера
