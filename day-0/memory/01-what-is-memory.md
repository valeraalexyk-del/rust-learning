# Что такое память?

[← Оглавление](README.md) | [Биты и байты →](02-bits-bytes.md)

---

## Простыми словами

Когда ты запускаешь программу, компьютеру нужно где-то хранить данные:
- Числа, которые ты вводишь
- Текст, который ты печатаешь
- Результаты вычислений

**Память (RAM)** — это место, где всё это хранится.

---

## Представь шкаф с ящиками

```
Память компьютера:
┌─────────┬─────────┬─────────┬─────────┐
│ Ящик 0  │ Ящик 1  │ Ящик 2  │ Ящик 3  │ ...
│ [    ]  │ [    ]  │ [    ]  │ [    ]  │
└─────────┴─────────┴─────────┴─────────┘
     ↑         ↑         ↑
  адрес 0   адрес 1   адрес 2
```

Каждый ящик имеет **номер** (адрес).

Чтобы найти данные — нужно знать номер ящика.

---

## Главное

- Память — место хранения данных
- У каждой ячейки есть адрес (номер)
- Программа записывает и читает данные по адресам
