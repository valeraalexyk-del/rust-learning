## Бит (bit) — самая маленькая единица

**Бит** (англ. **bit**, сокращение от **bi**nary digi**t** — двоичная цифра) может хранить только `0` или `1`.

```
Бит — как выключатель:

  0 = выкл      1 = вкл
  ○             ●
```

---

## Байт (byte) = 8 бит

**Байт** (англ. **byte**) — это 8 бит вместе.

```
Один байт:
┌───┬───┬───┬───┬───┬───┬───┬───┐
│ 0 │ 1 │ 1 │ 0 │ 0 │ 1 │ 0 │ 1 │
└───┴───┴───┴───┴───┴───┴───┴───┘
```

В одном байте можно хранить число от 0 до 255.

---

## Размеры

| Размер | Сколько | Пример |
|--------|---------|--------|
| 1 байт | 8 бит | Одна буква |
| 1 КБ | 1024 байта | Страница текста |
| 1 МБ | 1024 КБ | Фотография |
| 1 ГБ | 1024 МБ | Фильм |

---

## В Rust

```rust
let x: i8 = 42;   // 1 байт (8 бит)
let y: i32 = 42;  // 4 байта (32 бита)
let z: i64 = 42;  // 8 байт (64 бита)
```

Число в названии типа = размер в битах.
