# Дробные числа (Floating-Point)

## Что это?

**Дробные числа** (англ. **floating-point numbers**) — числа с точкой: `3.14`, `0.5`, `-2.7`

> **Floating-point** = с плавающей точкой. Точка "плавает" — может быть в разных местах: `3.14`, `31.4`, `0.314`.

---

## Типы

```
f64
│ │
│ └── 64 = размер в битах
└──── f = float (дробное, floating-point)
```

| Тип | Размер | Точность | Пример |
|-----|--------|----------|--------|
| `f32` | 4 байта | ~7 цифр | `3.1415927` |
| `f64` | 8 байт | ~15 цифр | `3.141592653589793` |

---

## Примеры

```rust
let pi: f64 = 3.14159265359;
let price: f32 = 19.99;

// По умолчанию Rust выбирает f64
let x = 3.14;  // это f64
```

---

## Почему f64 по умолчанию?

- `f64` точнее чем `f32`
- На современных компьютерах работает так же быстро
- Меньше ошибок округления

---

## Когда использовать?

| Ситуация | Тип |
|----------|-----|
| Цены, деньги | `f64` |
| Физика, математика | `f64` |
| Графика, игры | `f32` (быстрее на GPU) |
