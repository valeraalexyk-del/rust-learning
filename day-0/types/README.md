## Быстрая шпаргалка

```
СКАЛЯРНЫЕ ТИПЫ (Scalar Types) — одно значение:
─────────────────────────────────────────────
  i8, i16, i32, i64    — целые со знаком (signed integers)
  u8, u16, u32, u64    — целые без знака (unsigned integers)
  f32, f64             — дробные (floating-point)
  bool                 — логический (boolean)
  char                 — символ (character)

СОСТАВНЫЕ ТИПЫ (Compound Types) — несколько значений:
─────────────────────────────────────────────
  (T1, T2)             — кортеж (tuple)
  [T; N]               — массив (array)

СТРОКИ (Strings):
─────────────────────────────────────────────
  &str                 — строковый срез (string slice)
  String               — строка (heap-allocated string)
```
