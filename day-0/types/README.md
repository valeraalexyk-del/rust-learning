## Быстрая шпаргалка

```
ЧИСЛА:
  i8, i16, i32, i64    — целые со знаком (могут быть минус)
  u8, u16, u32, u64    — целые без знака (только плюс)
  f32, f64             — дробные (с точкой)

ДРУГИЕ:
  bool                 — да/нет (true/false)
  char                 — один символ ('A')

СОСТАВНЫЕ:
  (T1, T2)             — кортеж (разные типы)
  [T; N]               — массив (одинаковые типы)

СТРОКИ:
  &str                 — простая строка
  String               — изменяемая строка
```
