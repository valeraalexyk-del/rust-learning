# Что такое тип данных? (Data Type)

[← Оглавление](README.md) | [Целые числа →](02-integers.md)

---

## Простыми словами

Компьютер хранит ВСЁ как числа (нули и единицы).

**Тип** говорит компьютеру, как понимать эти числа.

---

## Пример

В памяти хранится число `65`. Что это?

```
Число 65 в памяти:
┌───┬───┬───┬───┬───┬───┬───┬───┐
│ 0 │ 1 │ 0 │ 0 │ 0 │ 0 │ 0 │ 1 │
└───┴───┴───┴───┴───┴───┴───┴───┘

Если тип "число":    65
Если тип "буква":    A  (код буквы A = 65)
```

**Одни и те же данные — разный смысл!**

---

## Зачем нужны типы?

| Причина | Пример |
|---------|--------|
| Понять данные | Это число или текст? |
| Знать размер | Сколько памяти занять? |
| Проверить ошибки | Нельзя сложить число и текст |

---

## Типы в Rust

```rust
let age = 25;           // целое число (integer, i32)
let price = 19.99;      // дробное (floating-point, f64)
let active = true;      // логический (boolean, bool)
let letter = 'A';       // символ (character, char)
let name = "Иван";      // строка (string slice, &str)
```

Rust сам определяет тип — это называется **вывод типа** (англ. **type inference**).

Но можно указать тип явно — это **аннотация типа** (англ. **type annotation**):

```rust
let age: i32 = 25;      // явно указали тип i32
//     ^^^^ — аннотация типа
```
