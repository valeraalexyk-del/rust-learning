# Целые числа (Integers)

[← Что такое тип?](01-what-is-type.md) | [Оглавление](README.md) | [Дробные числа →](03-floats.md)

---

## Что это?

**Целые числа** (англ. **integers**) — числа без точки: `1`, `42`, `-100`, `0`

---

## Названия типов

В Rust целые числа называются `i32`, `u8` и т.д.

```
i32
│ │
│ └── 32 = размер в битах
└──── i = может быть минус (integer)

u8
││
│└── 8 = размер в битах
└─── u = только плюс (unsigned = без знака)
```

---

## Со знаком (signed) — могут быть минус

> **Signed** = знаковый, может иметь знак минус.

```
Примеры: -100, -5, 0, 5, 100
```

| Тип | Размер | Диапазон | Для чего |
|-----|--------|----------|----------|
| `i8` | 1 байт | -128 ... 127 | Температура |
| `i16` | 2 байта | -32 768 ... 32 767 | Высота |
| `i32` | 4 байта | ≈ ±2 миллиарда | Обычные числа |
| `i64` | 8 байт | Огромные | Большие вычисления |

---

## Без знака (unsigned) — только плюс

> **Unsigned** = беззнаковый, не может быть отрицательным.

```
Примеры: 0, 1, 5, 100, 255
```

| Тип | Размер | Диапазон | Для чего |
|-----|--------|----------|----------|
| `u8` | 1 байт | 0 ... 255 | Возраст |
| `u16` | 2 байта | 0 ... 65 535 | Год |
| `u32` | 4 байта | 0 ... ~4 миллиарда | Количество |
| `u64` | 8 байт | Огромные | ID в базе |

---

## Как выбрать?

```
Вопрос: Число может быть отрицательным?

ДА  → используй i (i8, i16, i32, i64)
НЕТ → используй u (u8, u16, u32, u64)
```

---

## Примеры

```rust
// Температура бывает минусовой
let temp: i8 = -20;

// Возраст не бывает минусовым
let age: u8 = 25;

// По умолчанию Rust выбирает i32
let number = 42;  // это i32
```

---

## Визуально: размеры

```
i8  = 1 байт  = [████████]

i16 = 2 байта = [████████][████████]

i32 = 4 байта = [████████][████████][████████][████████]
```

Чем больше байт — тем большее число можно хранить.
