## Два вида строк

В Rust есть два типа строк:

```
&str    — строковый срез (string slice), нельзя менять
String  — строка в куче (heap-allocated), можно менять
```

---

## &str — строковый срез (String Slice)

```rust
let greeting: &str = "Привет!";
```

> **String slice** (строковый срез) — ссылка на последовательность символов. Неизменяемая.

- Нельзя изменить (immutable)
- Быстрая (не требует выделения памяти)
- Для текста, который не меняется
- Хранится в бинарнике программы или как ссылка на `String`

---

## String — динамическая строка

```rust
let mut name = String::from("Иван");
name.push_str(" Петров");
// теперь name = "Иван Петров"
```

> **String** — строка, выделенная в куче (heap). Может расти и изменяться.

- Можно изменять (mutable, если объявлена с `mut`)
- Владеет своими данными (owned)
- Для текста, который меняется

---

## Сравнение

| | `&str` | `String` |
|-|--------|----------|
| Изменяемость | Нет | Да |
| Создание | `"текст"` | `String::from("текст")` |
| Добавить текст | Нельзя | `push_str()` |

---

## Пример

```rust
// Простая строка — не меняется
let hello: &str = "Привет";

// Изменяемая строка
let mut message = String::from("Привет");
message.push_str(", мир!");
// message = "Привет, мир!"
```

---

## Когда что использовать?

| Ситуация | Тип |
|----------|-----|
| Текст не меняется | `&str` |
| Нужно добавлять/менять текст | `String` |
| Не знаешь — начни с | `&str` |
