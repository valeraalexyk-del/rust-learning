## Два вида строк

В Rust есть два типа строк:

```
&str    — простая, нельзя менять
String  — изменяемая, можно добавлять текст
```

---

## &str — простая строка

```rust
let greeting: &str = "Привет!";
```

- Нельзя изменить
- Быстрая
- Для текста, который не меняется

---

## String — изменяемая строка

```rust
let mut name = String::from("Иван");
name.push_str(" Петров");
// теперь name = "Иван Петров"
```

- Можно изменять
- Для текста, который меняется

---

## Сравнение

| | `&str` | `String` |
|-|--------|----------|
| Изменяемость | Нет | Да |
| Создание | `"текст"` | `String::from("текст")` |
| Добавить текст | Нельзя | `push_str()` |

---

## Пример

```rust
// Простая строка — не меняется
let hello: &str = "Привет";

// Изменяемая строка
let mut message = String::from("Привет");
message.push_str(", мир!");
// message = "Привет, мир!"
```

---

## Когда что использовать?

| Ситуация | Тип |
|----------|-----|
| Текст не меняется | `&str` |
| Нужно добавлять/менять текст | `String` |
| Не знаешь — начни с | `&str` |
