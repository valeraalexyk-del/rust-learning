## Что это?

**Составные типы** (англ. **compound types**) — несколько значений вместе.

В Rust два составных типа: **кортеж** (tuple) и **массив** (array).

---

## Кортеж (Tuple)

**Кортеж** (англ. **tuple**) — несколько значений РАЗНЫХ типов в одной "коробке".

### Создание

```rust
let person = (25, 175.5, 'M');
//            ↑    ↑     ↑
//         возраст рост  пол
```

### Доступ к элементам

```rust
let person = (25, 175.5, 'M');

let age = person.0;      // 25
let height = person.1;   // 175.5
let gender = person.2;   // 'M'
```

**Нумерация с нуля!**

```
person = (25, 175.5, 'M')
          │    │     │
          .0   .1    .2
```

---

## Массив (Array)

**Массив** (англ. **array**) — несколько значений ОДНОГО типа.

> **Index** (индекс) — номер элемента в массиве. Нумерация с нуля!

### Создание

```rust
let numbers = [10, 20, 30, 40, 50];
//             ↑   ↑   ↑   ↑   ↑
//         5 чисел одного типа (i32)
```

### Доступ к элементам

```rust
let numbers = [10, 20, 30, 40, 50];

let first = numbers[0];   // 10
let second = numbers[1];  // 20
let last = numbers[4];    // 50
```

**Нумерация с нуля!**

```
[10, 20, 30, 40, 50]
 [0] [1] [2] [3] [4]
```

---

## Сравнение

| | Кортеж `(...)` | Массив `[...]` |
|-|----------------|----------------|
| Типы элементов | Разные | Одинаковые |
| Пример | `(25, "Иван")` | `[1, 2, 3]` |
| Доступ | `data.0` | `data[0]` |
