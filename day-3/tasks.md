# День 3 — Условия

## Задание 1
Создай переменную `temp`.  
Если:
- `temp > 30` → «Жарко»  
- `temp > 10` → «Тепло»  
- иначе → «Холодно»

## Задание 2
Определи, чётное число или нечётное.  
Используй:
```rust
let result = if ... { ... } else { ... };
```
