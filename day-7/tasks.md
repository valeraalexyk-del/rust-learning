# День 7 — match

## Задание 1
Используй `match`, чтобы определить:
- 1 → «Один»  
- 2 → «Два»  
- 3 → «Три»  
- другое → «Неизвестно»  

## Задание 2
Создай enum:
```rust
enum Status {
    Ok,
    Error,
    Loading,
}
```
Используй `match`, чтобы вывести:
- Ok → «Готово»  
- Error → «Ошибка»  
- Loading → «Загрузка…»  

## Зачем это нужно
- понять, как работает сопоставление с образцом  
- научиться обрабатывать разные варианты  

## Где применяется
- обработка ошибок  
- состояния игры  
- логика меню
