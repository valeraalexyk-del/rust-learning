# День 2 — Функции

## Задание 1
Напиши функцию `hello()`, которая выводит:
```
Привет, мир!
```
Вызови её из `main`.

## Задание 2
Напиши функцию:
```rust
fn multiply(a: i32, b: i32) -> i32
```
Она должна возвращать произведение двух чисел.  
В `main` вызови её и выведи результат.

## Зачем это нужно
- понять, как объявлять функции  
- научиться передавать параметры  
- научиться возвращать значения  

## Где применяется
- разбиение программы на части  
- повторное использование кода  
- создание логики в проектах
