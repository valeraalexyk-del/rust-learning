# День 5 — Структуры

## Задание 1
Создай структуру:
```rust
struct Car {
    brand: String,
    year: u32,
}
```
Создай объект и выведи его поля.

## Задание 2
Добавь метод:
```rust
impl Car {
    fn age(&self) -> u32 {
        2025 - self.year
    }
}
```
Выведи возраст машины.

## Зачем это нужно
- понять, как создавать свои типы данных  
- научиться объединять данные в одну структуру  
- научиться писать методы  

## Где применяется
- модели данных  
- игровые объекты  
- конфигурации
